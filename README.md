# My Personal Portfolio Website

Welcome to my personal portfolio website! This site serves as a showcase of my skills and projects as a web developer. Built with HTML, CSS, and JavaScript, the website features a sleek and modern design to highlight my proficiency in front-end development. The website is fully responsive, ensuring a seamless user experience across various devices.

## Features

- **Responsive Design**: The website adapts to different screen sizes, providing a seamless experience on desktops, tablets, and mobile devices.
- **Projects Section**: Showcases a selection of my notable projects, complete with descriptions, technologies used, and live links to the projects hosted on platforms like GitHub.
- **Contact Section**: Enables visitors to reach out to me directly with questions or collaboration opportunities. Upon submission, the form sends the entered data to my Google Sheet.

## Contact
If you'd like to send me a message directly through the website, please use the contact form on the site. The form will send your message directly to my Google Sheet.

## Technologies Used

- **HTML**: For structuring the content on the web pages.
- **CSS**: For styling the website to make it visually appealing.
- **JavaScript**: For adding interactivity and dynamic content.

Thank you for visiting my portfolio website! I hope you enjoy exploring my work as much as I enjoyed creating it.
